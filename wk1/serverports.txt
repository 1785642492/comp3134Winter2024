What is the definition of server port?
the logical endpoints of a network connection that is used to exchange information between a web server and a web client

What is server port range?
0 to 65536

How many bits does number above represent?
The range of port numbers from 0 to 65535 can be represented using 16 bits. ​​

State 10 common server ports
Port 20 and 21: FTP (File Transfer Protocol) for data transfer.
Port 22: SSH (Secure Shell) for secure logins, file transfers, and port forwarding.
Port 23: Telnet protocol for unencrypted text communications.
Port 25: SMTP (Simple Mail Transfer Protocol) for email routing.
Port 53: DNS (Domain Name System) for translating domain names into IP addresses.
Port 80: HTTP (Hypertext Transfer Protocol) for web traffic.
Port 110: POP3 (Post Office Protocol version 3) for email retrieval.
Port 143: IMAP (Internet Message Access Protocol) for managing email messages.
Port 443: HTTPS (HTTP Secure) for secure web traffic.
Port 3306: MySQL database system.
